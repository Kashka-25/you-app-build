# Styles

The design system, split by concern. Loaded in index.html before any feature renders.

- tokens.css   — THE source of truth: colours, fonts, the reset. Change the theme here.
- auth.css     — auth + onboarding + level-up overlay
- layout.css   — app shell, avatar panel, header, xp bar, stats, tabs
- cards.css    — add form, filters, item cards
- calendar.css — memory, astrology, calendar, mood
- values.css   — tags, pillars, values
- modals.css   — modals, toast, and all mobile (@media) rules
